# Reflection – Library Management System

## Project Overview

This project is a Python Library Management System designed to demonstrate several programming concepts taught during the course. The system allows books to be stored, listed, borrowed, and returned. The program also allows members to register and borrow books. The application is built using object-oriented programming principles and includes features such as file storage, testing, and command-line interaction.

The main goal of the project was to demonstrate understanding of Python fundamentals including classes, inheritance, validation, file input/output, and automated testing.
